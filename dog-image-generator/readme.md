# 🐶 Dog Image Generator

🌐 Live Demo: https://ceo-dog-api.netlify.app/

A simple web application that fetches and displays random dog images using the Dog CEO API. Each click generates a new dog image dynamically.

---

## 🚀 Features

* 🐶 Fetch random dog images
* 🔁 Generate new image on each click
* ⚡ Fast API-based response
* 📱 Responsive design using Bootstrap
* ❌ Error handling for failed API requests

---

## 🛠️ Tech Stack

* HTML
* CSS (Bootstrap 5)
* JavaScript (Vanilla JS)
* Axios (API calls)
* Dog CEO API

---

## Concepts Used

* API Integration
* Asynchronous JavaScript (`async/await`)
* HTTP Requests using Axios
* DOM Manipulation
* Event Handling (`onclick`)

## 📌 How It Works

1. User clicks the **Click** button
2. App sends request to Dog CEO API
3. Receives random dog image URL
4. Displays the image dynamically on screen
5. Handles errors if API fails

---

## 🔗 API Used

* Dog CEO API: https://dog.ceo/dog-api/

---

## ⚠️ Limitations

* Requires internet connection
* Only shows one image at a time
* No image history
