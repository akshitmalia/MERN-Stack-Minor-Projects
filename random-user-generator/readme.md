# 👤 Random User Generator

🌐 Live Demo: https://random-user-generator-akshit.netlify.app/

A simple web application that fetches and displays detailed information about a random user using an external API. The app dynamically updates the UI to show user profile data including personal details, location, and contact information.

---

## 🚀 Features

* 👤 Generate random user data
* 📄 Display detailed profile information:

  * Name, Gender
  * Email, Phone, Cell
  * Address & Location
  * Date of Birth & Age
  * Registration details
* 🖼️ Show user profile image
* 🔄 Back button to return to home screen
* ⚡ Loading state while fetching data
* ❌ Error handling for failed API requests

---

## 🛠️ Tech Stack

* HTML
* CSS (Bootstrap 5)
* JavaScript (Vanilla JS)
* Axios (API calls)
* Random User API

---

## 🧠 Concepts Used

* DOM Manipulation

* Dynamic UI Rendering

## ⚙️ Installation & Setup

```
git clone https://github.com/akshitmalia/random-user-generator.git
cd random-user-generator
```

* Open `index.html` in your browser

---

## 📌 How It Works

1. User clicks the **Check** button
2. App sends request to Random User API
3. Receives random user data
4. Displays user details in a card format
5. User can click **Back** to return to home screen

---

## 🔗 API Used

* Random User API: https://randomuser.me/

---

## ⚠️ Limitations

* Requires internet connection
* Shows only one user at a time
* No data persistence
