# 🗣️ Joke Generator with Speech Synthesis

 🌐 Live Demo: https://joke-api-speech-synthesis-akshit.netlify.app/

An interactive web application that fetches random jokes from an API and reads them aloud using the Web Speech API. The app displays jokes and provides a voice-based experience for better user engagement.

---

## 🚀 Features

* 😂 Fetch random jokes from API
* 🗣️ Convert text to speech (Speech Synthesis)
* 🎭 Dialogue-style joke presentation (Laurel & Hardy)
* ⚡ Instant UI updates
* ❌ Error handling for API failures
* ⏹️ Stop speech on user interaction

---

## 🛠️ Tech Stack

* HTML
* CSS (Bootstrap 5)
* JavaScript (Vanilla JS)
* Axios (API calls)
* Web Speech API (Speech Synthesis)

---

##  Concepts Used

* API Integration
* Asynchronous JavaScript (`async/await`)
* DOM Manipulation
* Event Handling
* Speech Synthesis (`SpeechSynthesisUtterance`)
* Voice Control (pitch, rate, volume, language)

## ⚙️ Installation & Setup

```
git clone https://github.com/akshitmalia/joke-api-speech.git
cd joke-api-speech
```

* Open `index.html` in your browser

---

## 📌 How It Works

1. User clicks **Display**
2. App fetches a random joke from API
3. Displays:

   * Setup (Laurel)
   * Punchline (Hardy)
4. Converts text into speech using Speech API
5. Joke is spoken aloud automatically
6. Clicking anywhere stops the speech

---

## 🔊 Speech Customization

The app allows easy customization of speech properties:

* 🌐 Language (`lang`)
* ⚡ Speed (`rate`)
* 🎵 Pitch (`pitch`)
* 🔊 Volume (`volume`)

 These can be modified to control how the joke is spoken.
