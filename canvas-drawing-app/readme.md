# 🎨 Canvas Drawing App

🌐 Live Demo: https://akshit-canvas-drawing-app.netlify.app/

A simple and interactive drawing application built using the HTML5 Canvas API. Users can draw freely on the canvas with customizable brush size and color, clear the canvas, and export their artwork as a PNG image.

---

## 🚀 Features

* 🖌️ Freehand drawing on canvas
* 🎨 Custom brush color selection
* 📏 Adjustable brush size
* 🧹 Clear canvas functionality
* 💾 Save drawing as PNG image
* ⚡ Smooth real-time drawing experience
* 📱 Responsive canvas resizing

---

## 🛠️ Tech Stack

* HTML
* CSS (Bootstrap 5)
* JavaScript (Vanilla JS)
* HTML5 Canvas API

---

##  Concepts Used

* Canvas API (`getContext`, `lineTo`, `stroke`)
* Mouse Events (`mousedown`, `mousemove`, `mouseup`)
* DOM Manipulation
* Event Handling
* Dynamic Styling (color & brush size)
* Image Export (`toDataURL`)
* Responsive Canvas Resizing

## 📌 How It Works

1. User presses mouse (`mousedown`) to start drawing
2. Moves cursor to draw lines (`mousemove`)
3. Releases mouse to stop drawing (`mouseup`)
4. Brush color and size are applied dynamically
5. User can:

   * Clear canvas
   * Save drawing as PNG

---

## 📱 Responsiveness

* Canvas automatically resizes based on screen width
* Maintains drawing area consistency across devices

---

## ⚠️ Limitations

* No undo/redo functionality
* Drawing is lost on refresh
* No touch support for mobile devices
